A Quick Note on Firebase Integration:
When you take the code generated from this prompt, you will need to wire up the actual Firebase backend. Keep in mind:

Email Verification: Firebase Auth has a built-in sendEmailVerification function, but out-of-the-box, it sends a link that the user clicks, rather than a 6-digit code. If you strictly want a 6-digit code UI, you will need to use Firebase Cloud Functions combined with a service like SendGrid or Nodemailer to generate, send, and verify that specific code.

Google Login: You'll need to enable the Google Sign-In provider in your Firebase Console under Authentication > Sign-in method.
